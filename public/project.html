MY PROJECT /私のプロジェクト

Here are some works I have designed and developed

School Management System Website -Team Project

This School Management System is a web-based platform built with Node.js and Tailwind CSS, designed to enhance communication and streamline operations between students, parents, teachers, and school staff. It features role-based access, allowing users to manage profiles, access academic data, share learning materials, track student progress, and participate in school activities efficiently.

E-Commerce Fashion Shop Website

T4K3Shi is a men’s fashion e-commerce website built with HTML, Tailwind CSS, JavaScript, PHP, and MySQL. It features category-based product pages, a shopping cart, checkout system, and an admin dashboard for inventory management. The design focuses on clean UI, responsive layout, and a stylish user experience.

Restaurant Website Page Design

FoodFusion is a community-driven culinary website built with PHP, MySQL, JavaScript, and HTML, Bootstrap CSS, designed to promote home cooking, recipe sharing, and food education. It features a responsive design with categorized recipes, user submissions, cooking tutorials, and educational resources on culinary and environmental topics.
